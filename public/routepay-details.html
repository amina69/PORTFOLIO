About The Project

The project aimed to developing an admin dashboard for monitoring QR code transactions for a Fintech company (RoutePay) that specialises in innovative payment solutions and services (while working for a Tech Solutions Company).

The web application featured a comprehensive dashboard displaying performance metrics and transaction reports, QR code management tools, merchant and admin user management functionalities, audit trails, and activity logs.

Problem Statement
RoutePay, a fintech company specializing in innovative payment solutions, required an efficient system to monitor and manage QR code transactions. Existing tools lacked the functionality to provide real-time insights, transaction tracking, and user management in a unified interface, hindering the company's ability to maintain operational efficiency and ensure accountability. The need for a comprehensive admin dashboard became evident to streamline monitoring, enhance management capabilities, and ensure scalability for growing user demands.
Research Methods
Conducted interviews with RoutePay's management, merchants, and admin users to identify key requirements and pain points.
Analyzed similar fintech dashboards to identify industry standards and best practices.
Implemented an iterative design process, incorporating user feedback to refine the dashboard's features and user interface..
Findings
Admin users needed robust tools for managing accounts, permissions, and QR codes.
Audit trails and activity logs were critical for maintaining transparency and tracking actions taken within the system.
The ability to access real-time data on QR code transactions was crucial for decision-making and performance evaluation.
